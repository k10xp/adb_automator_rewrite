# Frontend GUI

## Setup

Install dependencies before starting development or building:

```shell
npm install
```

## Development

Start the development server with hot reload:

```shell
npm run dev
```

## Build

Create a production build:

```shell
npm run build
```

## Preview

Serve the production build locally:

```shell
npm run preview
```

## Electron

```shell
npm run dev:electron
```
